# I Knew Maths Once (Ediția originală în Limba Română)

Acest depozit reprezintă o arhivă digitală a caietelor mele de Matematică Pură din perioada studenției la Universitatea din București, începând cu programul de Algebră din octombrie 1994.

Ca parte a inițiativei "Maths from the Attic", acest proiect transcrie în LaTeX teoremele, demonstrațiile și cursurile de algebră exact așa cum au fost predate și notate la acea vreme.

După o tranziție de la rigoarea matematicii pure către o carieră de peste 25 de ani ca BI & Web Developer, acest proiect îmbină logica matematică fundamentală cu structurile moderne de versionare a codului. Arhiva este menținută prin sincronizarea directă dintre Overleaf și GitHub.

## Structura Proiectului
* `courses/`: Notițele teoretice (Cursuri).
* `seminars/`: Problemele aplicate și exercițiile (Seminarii).
